# Srm / Some Ruby Methods

- A collection of helpful Ruby methods
- This is a prototype
- Build w/ Ruby 2.2.3
- Tested w/ Rspec 3.4

## Installation

Install on your system:

```
gem install srm
```

In your app:

```
# Add this to your Gemfile
gem 'srm'
```
And then run `bundle install`
